# 简介

以语音形式进行交互的自动问答系统。程序分为客户端和服务器端。客户端为安卓 APP.
FAQ 模块主要使用**基于语义融合的 VSM 算法**进行答案过滤,过滤出 10 个具可能答案。
然后使用**朴素贝叶斯分类器**进行答案抽取。语料库为在线爬取百度知道的答案和 360 文库的文本。
